# Peripheral's Access

## Synopsis

```
$ rpio peripheral help
arguments: MODE [help]
MODE | arm-timer   # free running ARM counter
     | cm          # clock manager
     | dma         # direct memory access
     | gpio        # general purpose input/output
     | intr        # interrupt control
     | mbox        # mailbox control
     | pwm         # pulse width modulation
     | spi0        # serial peripheral interface:0
     | spi1        # serial peripheral interface:1
     | spi-slave   # serial peripheral interface:slave
```
