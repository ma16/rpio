# Serial Peripheral Interface (SPI1)

## Synopsis

```
$ ./rpio spi1 help
arguments: MODE [help]

MODE | control  # set control registers
     | peek     # peek word from rx-fifo
     | rate     # throughput test
     | read     # read word from rx-fifo
     | status   # display control and status registers
     | write    # write word to tx-fifo
     | xfer     # transceive multiple words
```
